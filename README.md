# EDS 232 – In-Class Materials

In-class notebooks and code-alongs for **EDS 232: Machine Learning for Environmental Science** at UC Santa Barbara's Bren School of Environmental Science & Management.

## Contents

Work completed during lecture, including annotated notebooks, demonstrations, and exploratory code covering topics like statistical learning, model accuracy, linear regression, and other ML methods introduced in class.

## Setup

This repo uses the `eds232-env` conda environment. To build it:
```bash
conda env create --name eds232-env --file eds232-env.yml
conda activate eds232-env
```

Then register the Jupyter kernel:
```bash
python -m ipykernel install --user --name eds232-env --display-name "eds232-env"
```

See the [course setup page](https://meds-eds-232.github.io/EDS-232-course-website/notes/setup/setup.html) for full instructions.
